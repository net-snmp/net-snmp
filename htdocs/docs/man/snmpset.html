NAME

snmpset - communicates with a network entity using SNMP SET requests  

SYNOPSIS

snmpset [COMMON OPTIONS] OID TYPE VALUE [OID TYPE VALUE]...  

DESCRIPTION

snmpset is an SNMP application that uses the SNMP SET request to set information on a network entity. One or more object identifiers (OIDs) must be given as arguments on the command line. A type and a value to be set must accompany each object identifier. Each variable name is given in the format specified in dpz@noc.rutgers.edu ip.ipforwarding.0 = 2

will set the variables sysContact.0 and ipForwarding.0:

system.sysContact.0 = STRING: "dpz@noc.rutgers.edu"
ip.ipForwarding.0 = INTEGER:
